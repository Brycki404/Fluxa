# Signal

Yield-safe signal implementation for event handling, similar to RBXScriptSignal.

## Key Features
- Connect/disconnect handlers
- Fire events with arguments
- Wait for events

## API
- `Signal.new()`
- `Signal:Connect(fn)`
- `Signal:DisconnectAll()`
- `Signal:Fire(...)`
- `Signal:Wait()`
- `Signal:Once(fn)`

See source for details and usage.
