# UniversalJointWriter

Applies poses to arbitrary rigs with Motor6Ds and Bones. Builds joint maps and retargets pose data.

## Key Features
- Build joint and retarget maps from a Model
- Apply pose data to a rig

## API
- `UniversalJointWriter.BuildJointMap(root)`
- `UniversalJointWriter.ApplyPose(jointMap, retarget, pose)`

See source for details and advanced usage.
