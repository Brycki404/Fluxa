# Fluxa

![Logo](docs/images/FLUXA_Transparent_White.png)

Fluxa is a custom runtime animation engine for Roblox and a replacement for the default Animator/AnimationController flow when you want full control over blending, layering, and replication behavior.

## Status

Fluxa is actively developed. Expect iterative API and feature updates.

## Install

### Wally

Add Fluxa from Wally:

[brycki404/fluxa](https://wally.run/package/brycki404/fluxa)

### Studio

Use the uncopylocked dev place for the latest module and demo content:

[Fluxa Dev Place](https://www.roblox.com/games/107928648886650/Fluxa-Dev)
